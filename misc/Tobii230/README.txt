Running the example, tobii_lsl.py
=================================

The example app provided here, tobii_lsl.py is an example of how to use the Tobii Pro SDK with LSL.

To run this example, you will need to install python 2 or 3 and pylsl. 

Using pip: python -m pip install pylsl

Then, after calibrating the eyetracker, open a terminal, navigate to this directory and run: python tobii_lsl.py
